# サイト公開

ローカルの変更をGitHub Pagesに反映する。

## 手順

1. `git status` で変更ファイルを確認する

2. 変更がない場合は「変更はありません」と報告して終了する

3. 変更がある場合：
   - 変更内容を確認し、簡潔な日本語のコミットメッセージを作成する
   - 変更ファイルをステージングする（.env、credentials等の機密ファイルは除外）
   - コミットする
   - `git push origin master` でプッシュする

4. GitHub Pagesのビルド状態を確認する（`gh api repos/yamadar-star/p-a469c704ddc2/pages/builds --jq '.[0] | {status, created_at}'`）

5. 結果を報告する：
   - 反映したファイル一覧
   - コミットメッセージ
   - サイトURL: https://yamadar-star.github.io/p-a469c704ddc2/
   - ビルド状態
